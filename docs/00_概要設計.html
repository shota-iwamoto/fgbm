F-GBM 2.0

Financial Ordered Robust Student-t Gradient Boosting Machine

LightGBM型ヒストグラム木、CatBoost型ordered boosting、 Student-tロバスト尤度、era-level distributionally robust optimization、 因子中立化、turnover正則化を統合した金融特化型GBDTの数学仕様である。

本設計は、あらゆる金融分布に対する普遍的な「過学習の完全排除」を保証するものではない。 分布外予測に対する無条件の保証はNo-Free-Lunch原理上不可能である。 本設計が与えるのは、過学習経路を明示的に制約し、目的関数の単調減少を検証でき、 既知のリーク・外れ値・regime集中・因子露出・turnoverに対する耐性を体系化した学習方式である。

2. データ、時系列情報集合、予測可能性

観測を次で表す。

\[ \mathcal D = \left\{ (x_i,c_i,z_i,y_i,e_i,a_i,\omega_i,\mathcal I_i) \right\}_{i=1}^{N}. \]
  • \(x_i\)：数値特徴
  • \(c_i\)：カテゴリ特徴
  • \(z_i\in\mathbb R^K\)：既知risk factor
  • \(y_i\)：回帰値、クラス、またはrelevance
  • \(e_i\)：era
  • \(a_i\)：instrument
  • \(\omega_i\geq0\)：サンプル重み
  • \(\mathcal I_i\)：予測時点で利用可能な情報集合

特徴量には必ず予測可能性条件を課す。

\[ (x_i,c_i,z_i) \quad\text{is measurable with respect to}\quad \mathcal I_i. \]

target horizonを \(h_i\) とすると、学習サンプルの目的変数が確定する時点は \(t_i+h_i\) である。ordered statisticに利用できる過去集合は、

\[ \mathcal P_i = \left\{ j: t_j+h_j+\delta_{\mathrm{embargo}}<t_i \right\} \]

とする。単に \(e_j<e_i\) だけでは、overlapping targetによるリークを防げない。

2.1 era内正規化

\[ W_e = \sum_{i:e_i=e}\omega_i, \qquad a_i = \frac{\omega_i}{W_{e_i}}, \qquad \sum_{i:e_i=e}a_i=1. \]

基本損失はera内で正規化し、eraサイズの違いを学習重要度から切り離す。

3. 加法木モデル

\[ F^{(M)}(x,c) = F^{(0)}(x,c) + \sum_{m=1}^{M}\alpha_m f_m(x,c), \] \[ f_m(x,c) = \sum_{j=1}^{J_m}v_{mj} \mathbf 1[(x,c)\in R_{mj}]. \]

\(f_m\) はヒストグラムベースの決定木である。 LightGBM型leaf-wise成長を使用できるが、次の制約を必須とする。

  • 最大深さ
  • 最小総曲率
  • 最小era effective coverage
  • カテゴリ統計のstrict-past制約
  • robust split lower-gain制約
  • 木追加後のexact objective line search

4. Student-t回帰objectiveの厳密化

4.1 階層表現

残差を \(r_i=y_i-F_i\) とする。

\[ r_i\mid\lambda_i \sim \mathcal N \left( 0,\frac{\sigma_i^2}{\lambda_i} \right), \qquad \lambda_i \sim \operatorname{Gamma} \left( \frac{\nu}{2},\frac{\nu}{2} \right). \]

周辺分布は、

\[ r_i\sim t_\nu(0,\sigma_i) \]

であり、定数項を含む負の対数尤度は、

\[ \ell_{\mathrm t}(r_i;\nu,\sigma_i) = \log\sigma_i + \frac12\log(\nu\pi) + \log\Gamma\left(\frac{\nu}{2}\right) - \log\Gamma\left(\frac{\nu+1}{2}\right) + \frac{\nu+1}{2} \log \left( 1+\frac{r_i^2}{\nu\sigma_i^2} \right). \]

4.2 勾配、厳密Hessian、影響関数

\[ g_i^{\mathrm t} = \frac{\partial\ell_{\mathrm t}}{\partial F_i} = -\frac{(\nu+1)r_i} {\nu\sigma_i^2+r_i^2}, \] \[ h_{i,\mathrm{exact}}^{\mathrm t} = \frac{\partial^2\ell_{\mathrm t}}{\partial F_i^2} = \frac{ (\nu+1)(\nu\sigma_i^2-r_i^2) }{ (\nu\sigma_i^2+r_i^2)^2 }. \]

影響関数に対応するscoreは、

\[ \psi(r) = \frac{(\nu+1)r}{\nu\sigma^2+r^2}. \]

したがって、

\[ \sup_r|\psi(r)| = \frac{\nu+1}{2\sqrt{\nu}\sigma}, \qquad \lim_{|r|\to\infty}\psi(r)=0. \]

Student-tは有界かつredescendingな影響関数を持つ。 一方で、redescending lossは大きな初期残差を過度に無視する危険があるため、 Huber warm startまたは自由度continuationを標準仕様とする。

4.3 MM majorizerの導出

\(\log(1+u)\) は \(u\geq0\) 上で凹関数であるため、 現在値 \(u_i^{(m)}\) における接線は大域的上界になる。

\[ \log(1+u) \leq \log(1+u_0) + \frac{u-u_0}{1+u_0}. \]

\(u=r^2/(\nu\sigma^2)\) を代入すると、 固定された \(\nu,\sigma\) の下で、

\[ \ell_{\mathrm t}(r) \leq C(r^{(m)}) + \frac12 w_i^{(m)}r^2, \] \[ w_i^{(m)} = \frac{\nu+1} {\nu\sigma_i^2+(r_i^{(m)})^2}. \]

したがってMM曲率は、

\[ h_{i,\mathrm{MM}}^{\mathrm t} = w_i^{(m)} >0. \]
これは負の厳密Hessianを便宜的に切り上げたものではない。 固定scale・固定自由度のStudent-t負対数尤度に対する正当な大域的quadratic majorizerである。

数値安定化後の曲率は、

\[ \bar h_i = \min \left[ h_{\max}, \max(h_{\min},h_{i,\mathrm{MM}}^{\mathrm t}) \right]. \]

4.4 scaleと自由度の識別

制約付きパラメータ化を用いる。

\[ \sigma_e = \sigma_{\min} + \operatorname{softplus}(s_e), \] \[ \nu = \nu_{\min} + (\nu_{\max}-\nu_{\min}) \operatorname{sigmoid}(\theta_\nu). \]

金融リターンの有限分散をモデル上要求する場合は \(\nu_{\min}>2\) とする。 redescending robustnessを優先し、有限分散を必須としない場合は \(\nu_{\min}>0\) でも定義可能である。

era scaleには階層正則化を加える。

\[ \mathcal R_\sigma = \frac{\lambda_\sigma}{2} \sum_e \left( \log\sigma_e-\log\sigma_0 \right)^2 + \frac{\lambda_{\sigma,t}}{2} \sum_{e=2}^{E} \left( \log\sigma_e-\log\sigma_{e-1} \right)^2. \]

木更新中は \(\sigma_e,\nu\) を固定し、 木を更新した後にscale・自由度を更新するblock coordinate方式を採用する。 \(\sigma_{\min}>0\) によりscale collapseを防止する。

5. Capped KL Era-DRO

5.1 era損失

\[ L_e(F) = \sum_{i:e_i=e} a_i\ell_i(F_i). \]

基準era分布を \(\pi_e>0\)、\(\sum_e\pi_e=1\) とする。 通常は均等重み、または時間減衰を正規化した重みを用いる。

5.2 ambiguity set

\[ \mathcal Q_{\rho,\kappa} = \left\{ q\in\mathbb R_+^E: \sum_eq_e=1,\; D_{\mathrm{KL}}(q\Vert\pi)\leq\rho,\; q_e\leq\kappa\pi_e \right\}. \]

ロバストriskは、

\[ \boxed{ \mathcal R_{\mathrm{DRO}}(F) = \max_{q\in\mathcal Q_{\rho,\kappa}} \sum_eq_eL_e(F) } \]

である。\(\kappa=\infty\) なら通常のKL-DROに戻る。 capは破損eraへの敵対重み集中を抑制する。

5.3 capなしの場合の双対

\[ \mathcal R_{\mathrm{KL}}(F) = \inf_{\tau>0} \left\{ \tau\rho + \tau \log \left[ \sum_e \pi_e \exp(L_e(F)/\tau) \right] \right\}. \]

制約付きKL-DROでは \(\tau\) を最適化しなければならない。 固定した \(\tau\) を使う場合は、次の別objectiveである。

\[ \mathcal R_{\mathrm{entropic},\tau}(F) = \tau \log \left[ \sum_e\pi_e\exp(L_e(F)/\tau) \right].

5.4 DRO勾配

最適解 \(q^\star\) が一意なら、envelope theoremにより、

\[ \frac{\partial\mathcal R_{\mathrm{DRO}}}{\partial F_i} = q_{e_i}^\star a_i \frac{\partial\ell_i}{\partial F_i}. \]

一意でない場合も、任意の最適 \(q^\star\) が有効な部分勾配を与える。 \(q^\star\) は各木の開始時に凸最適化問題として解く。

6. Era安定性の再設計

6.1 固定benchmarkに対するdownside risk

era平均との差を二乗すると、平均自体がモデル依存で複雑な非凸結合を生む。 代わりに、学習前に固定したbenchmark \(b_e\) に対する悪化だけを罰する。

\[ d_e(F) = s_d \operatorname{softplus} \left( \frac{L_e(F)-b_e}{s_d} \right), \] \[ \mathcal R_{\mathrm{down}}(F) = \sum_e\pi_e d_e(F)^2. \]

\(b_e\) はzero model、linear baseline、または学習前に固定されたbenchmark lossとする。 現在モデルの平均損失を \(b_e\) に使ってはならない。

6.2 Era-CVaR

\[ \operatorname{CVaR}_\alpha(L) = \min_{\zeta\in\mathbb R} \left[ \zeta + \frac{1}{1-\alpha} \sum_e \pi_e (L_e-\zeta)_+ \right]. \]

微分可能版では、

\[ (u)_+ \approx s_c\operatorname{softplus}(u/s_c). \]

KL-DROとCVaRは共にtail-eraを重視するため、 両方を強く設定すると二重に保守的になる。 標準構成ではcapped KL-DROを主軸とし、 downside penaltyは小さな補助項とする。

6.3 任意のera分散項

\[ \mathcal R_{\mathrm{var}} = \sum_e\pi_e \left( L_e-\sum_k\pi_kL_k \right)^2. \]

この式自体は数学的に定義されるが、一般には凸でない。 使用する場合は、木構築時にGauss-Newton近似を使い、 後述するexact objective line searchを必須とする。 デフォルトでは無効とする。

7. Factor-Neutral PenaltyのPSD化

7.1 加重中心化

era \(e\) の予測ベクトルを \(F_e\)、 対角重み行列を \(W_e\) とする。

\[ C_e = I - \mathbf 1 \frac{ \mathbf 1^\top W_e }{ \mathbf 1^\top W_e\mathbf 1 }. \]

因子行列も中心化する。

\[ \widetilde Z_e=C_eZ_e. \]

7.2 因子射影

\[ M_e = \widetilde Z_e^\top W_e\widetilde Z_e + \gamma_ZI, \] \[ P_e = \widetilde Z_e M_e^{-1} \widetilde Z_e^\top W_e. \]

7.3 正定値二次ペナルティ

\(s_{F,e}>0\) を学習開始前または各木開始時に固定したscaleとする。 当該木の最適化中は微分しない。

\[ \boxed{ \mathcal R_{\mathrm{factor}}(F) = \frac12 \sum_e \frac{\pi_e}{s_{F,e}^2} (C_eF_e)^\top W_eP_e (C_eF_e) } \]

\(P_e\) は \(W_e\)-内積に関する射影であり、 各項は非負である。勾配は、

\[ \nabla_{F_e}\mathcal R_{\mathrm{factor}} = \frac{\pi_e}{s_{F,e}^2} C_e^\top W_eP_eC_eF_e. \]

Hessianは、

\[ H_{\mathrm{factor},e} = \frac{\pi_e}{s_{F,e}^2} C_e^\top W_eP_eC_e \succeq0. \]
旧版の予測分散で割るratio penaltyを廃止したため、 分母消失、scale縮小による不安定化、ratio由来の負曲率を回避できる。

7.4 Hard neutralizationとの区別

推論時の任意の後処理として、

\[ F_e^\perp = C_eF_e - \alpha_{\mathrm{neut}}P_eC_eF_e, \qquad 0\leq\alpha_{\mathrm{neut}}\leq1 \]

を使用できる。ただし学習時penaltyと推論時hard neutralizationは別操作であり、 validationでは両者を分離して評価する。

8. Turnover正則化の凸化

8.1 固定portfolio operator

era \(e\) のraw predictionからポジションスコアへの写像を、

\[ p_e=B_eF_e \]

とする。\(B_e\) は、中心化、ex-ante volatility scaling、 factor projection、流動性上限の線形近似を含む固定行列である。 \(B_e\) は当該boosting反復中に予測値へ依存させない。

8.2 instrument対応行列

連続era間で共通instrumentを対応させる行列を \(S_{e,e-1}\) とする。

\[ d_e = p_e-S_{e,e-1}p_{e-1}. \]

8.3 Smooth-L1 turnover

\[ \phi_\delta(u) = \sqrt{u^2+\delta^2}-\delta, \] \[ \boxed{ \mathcal R_{\mathrm{turn}}(F) = \sum_{e=2}^{E} \sum_a c_{a,e}\, \phi_\delta(d_{a,e}) } \]

\(\phi_\delta\) は凸であり、

\[ \phi_\delta'(u) = \frac{u}{\sqrt{u^2+\delta^2}}, \qquad \phi_\delta''(u) = \frac{\delta^2} {(u^2+\delta^2)^{3/2}} \geq0. \]

したがって、固定された線形 \(B_e\) の下で \(\mathcal R_{\mathrm{turn}}\) は予測ベクトルに関して凸である。 \(\tanh\) と予測依存gross normalizationは学習目的から除外し、 必要なら推論後のportfolio optimizerで処理する。

9. 完全統合objective

\[ \boxed{ \begin{aligned} \mathcal J(F,\sigma,\nu) ={}& \max_{q\in\mathcal Q_{\rho,\kappa}} \sum_eq_eL_e(F;\sigma,\nu) \\ &+ \lambda_{\mathrm{down}} \mathcal R_{\mathrm{down}}(F) + \lambda_{\mathrm{factor}} \mathcal R_{\mathrm{factor}}(F) \\ &+ \lambda_{\mathrm{turn}} \mathcal R_{\mathrm{turn}}(F) + \lambda_\sigma\mathcal R_\sigma(\sigma) \\ &+ \lambda_{\mathrm{var}} \mathcal R_{\mathrm{var}}(F) + \sum_{m=1}^{M}\Omega(f_m). \end{aligned} } \]

木の構造正則化は、

\[ \Omega(f_m) = \gamma_{\mathrm{leaf}}J_m + \gamma_{\mathrm{depth}}D_m^2 + \sum_{A\in\operatorname{Leaves}(f_m)} \left[ \lambda_1|v_A| + \frac12 \lambda_{A,\mathrm{total}}v_A^2 \right]. \]

\(\lambda_{A,\mathrm{total}}\) は通常のL2正則化と、 後述するera coverage priorを含む。

10.1 Smooth quantile

\[ \ell_{\tau,\varepsilon}(r) = \tau\varepsilon \log(1+\exp(r/\varepsilon)) + (1-\tau)\varepsilon \log(1+\exp(-r/\varepsilon)). \]

\(s=\operatorname{sigmoid}(r/\varepsilon)\) とすると、

\[ g = 1-\tau-s, \qquad h = \frac{s(1-s)}{\varepsilon}. \]

10.2 Student-t continuation

初期木で外れ値を完全に無視しないため、

\[ \nu_m = \nu_{\mathrm{final}} + (\nu_{\mathrm{start}}-\nu_{\mathrm{final}}) \exp(-m/\tau_\nu), \qquad \nu_{\mathrm{start}}\gg\nu_{\mathrm{final}} \]

とする。初期はGaussianに近く、学習が進むにつれて裾を厚くする。

11. Student-t分類

11.1 Binary Student-t link

\[ y_i = \mathbf1[F_i+\epsilon_i>0], \qquad \epsilon_i\sim t_\nu(0,s). \]
\[ p_i = T_\nu(F_i/s), \qquad \ell_i = -y_i\log p_i -(1-y_i)\log(1-p_i). \]

\(f_\nu\) を標準Student-t密度とし、

\[ p_i' = \frac1s f_\nu(F_i/s). \]

勾配は、

\[ g_i = \frac{(p_i-y_i)p_i'} {p_i(1-p_i)}. \]

厳密Hessianは観測ラベルに依存し、尾部で不安定になり得る。 木構築にはFisher曲率を使う。

\[ h_{i,\mathrm F} = \frac{(p_i')^2} {p_i(1-p_i)} + h_{\min}. \]

数値計算ではStudent-t CDFを直接引いて \(1-p\) を求めず、log-CDFとlog-survival functionを使う。

11.2 ラベル汚染モデル

\[ \widetilde p_i = (1-\epsilon_+-\epsilon_-)p_i+\epsilon_-, \] \[ 0\leq\epsilon_+,\epsilon_-, \qquad \epsilon_++\epsilon_-<1. \]

\(\epsilon_+\) と \(\epsilon_-\) はvalidationまたは階層priorで制約し、 無制約に学習して分類signalを吸収させてはならない。

11.3 Nominal multiclass

nominal classには、順序依存するstick-breakingを標準方式にしない。 数学的に対称なmultivariate Student-t random utility modelを定義する。

\[ U_{ik} = F_{ik}+\epsilon_{ik}, \qquad \epsilon_i = \frac{L\xi_i}{\sqrt{\lambda_i}}, \] \[ \xi_i\sim\mathcal N(0,I), \qquad \lambda_i\sim \operatorname{Gamma} \left( \frac\nu2,\frac\nu2 \right), \] \[ y_i=\arg\max_kU_{ik}. \]

クラス確率は、

\[ p_{ik} = P \left( U_{ik}\geq U_{ij} \;\forall j\neq k \right). \]

一般に閉形式はないため、準Monte Carloまたは共通乱数付きMonte Carloで近似する。 大規模用途では、対称なone-vs-rest Student-t linkを正規化した近似モードを提供するが、 これは厳密random utility likelihoodとは区別する。

12. Student-tランキング

12.1 共有scale-mixtureによる正当化

queryまたはera \(e\) 内で、共通混合変数を仮定する。

\[ \lambda_e \sim \operatorname{Gamma} \left( \frac\nu2,\frac\nu2 \right), \] \[ \epsilon_i\mid\lambda_e \sim \mathcal N \left( 0,\frac{\sigma_e^2}{\lambda_e} \right). \]

\(i,j\) が同一query-eraに属するとき、

\[ \epsilon_j-\epsilon_i \mid\lambda_e \sim \mathcal N \left( 0,\frac{2\sigma_e^2}{\lambda_e} \right), \] \[ \epsilon_j-\epsilon_i \sim t_\nu(0,\sqrt2\sigma_e). \]
独立な周辺Student-t変数の差が一般にStudent-tになるわけではない。 共有Gamma混合変数を導入することで、pairwise Student-t linkを数学的に正当化する。

12.2 Pairwise likelihood

\[ d_{ij}=F_i-F_j, \qquad p_{ij} = T_\nu \left( \frac{d_{ij}}{\sqrt2\sigma_e} \right). \]
\[ \ell_{ij} = -b_{ij}\log p_{ij} -(1-b_{ij})\log(1-p_{ij}). \]

\(s_e=\sqrt2\sigma_e\)、 \(p_{ij}'=f_\nu(d_{ij}/s_e)/s_e\) とすると、

\[ g_{ij}^{(d)} = \frac{ (p_{ij}-b_{ij})p_{ij}' }{ p_{ij}(1-p_{ij}) }, \] \[ h_{ij}^{(d)} = \frac{(p_{ij}')^2} {p_{ij}(1-p_{ij})} + h_{\min}. \]

サンプルへの集約は、

\[ g_i\mathrel{+}=w_{ij}g_{ij}^{(d)}, \qquad g_j\mathrel{-}=w_{ij}g_{ij}^{(d)}, \] \[ h_i\mathrel{+}=w_{ij}h_{ij}^{(d)}, \qquad h_j\mathrel{+}=w_{ij}h_{ij}^{(d)}. \]

12.3 Pair sampling

全pairは \(O(n_e^2)\) なので、目的を変えないunbiased samplingを使う。 pair抽出確率を \(\pi_{ij}^{\mathrm{pair}}>0\) とすると、

\[ \widehat L_e^{\mathrm{rank}} = \frac1{M_e} \sum_{(i,j)\in\mathcal S_e} \frac{ w_{ij}\ell_{ij} }{ \pi_{ij}^{\mathrm{pair}} } \]

は定数倍を除いて全pair損失の不偏推定量になる。

12.4 NDCGモード

次の重みを各boosting反復の開始時に固定する。

\[ w_{ij}^{(m)} = \left| \Delta\mathrm{NDCG}_{ij} (F^{(m-1)}) \right|. \]

これは固定されたpairwise surrogateに対しては正当な最適化になるが、 反復間で重みが変わるため、全反復を通じた単一固定objectiveとは限らない。 よって、次を区別する。

  • student_t_pairwise：固定pair重みを持つ厳密likelihood objective
  • student_t_lambdarank：反復ごとにNDCG重みを固定するLambda surrogate

13. Ordered Categorical Statistics

13.1 Strict information-set rule

sample \(i\) のカテゴリ統計には \(\mathcal P_i\) の観測だけを使う。

\[ \operatorname{OTS}_{i,c} = \frac{ a_c\mu_0 + \sum_{j\in\mathcal P_i} K(t_i,t_j) \omega_j \mathbf1[c_j=c_i] \phi(y_j) }{ a_c + \sum_{j\in\mathcal P_i} K(t_i,t_j) \omega_j \mathbf1[c_j=c_i] }. \]

時間kernelは、

\[ K(t_i,t_j) = \exp[-\lambda_t(t_i-t_j)] \mathbf1[t_j+h_j+\delta_{\mathrm{embargo}}<t_i]. \]

13.2 Student-t robust category location

カテゴリ内locationをIRLSで推定する。

\[ \lambda_j^{(k)} = \frac{\nu_c+1} { \nu_c+ (\phi(y_j)-\mu_c^{(k)})^2/\sigma_c^2 }, \] \[ \mu_c^{(k+1)} = \frac{ a_c\mu_0 + \sum_{j\in\mathcal P_i} K_{ij}\omega_j \lambda_j^{(k)} \phi(y_j) }{ a_c + \sum_{j\in\mathcal P_i} K_{ij}\omega_j \lambda_j^{(k)} }. \]

\(\mu_c\) を両辺に含むため、旧版の式は一回の代入だけでは定義が完結しなかった。 2.0では固定点反復またはEMとして明示する。

13.3 Ordered boostingの実現

時系列foldを、

\[ \mathcal T_k = \{i:t_i+h_i+\delta_{\mathrm{embargo}}<A_k\}, \]
\[ \mathcal V_k = \{i:A_k\leq t_i<B_k\} \]

とし、\(\mathcal T_k\) だけで学習したモデルから、

\[ \widetilde F_i = F^{(-k)}(x_i,c_i), \qquad i\in\mathcal V_k \]

を生成する。ordered residualは、

\[ \widetilde g_i = \left. \frac{\partial\ell(y_i,F)} {\partial F} \right|_{F=\widetilde F_i} \]

とする。単一のin-sample model predictionからカテゴリ統計や残差を作らない。

14. 合成勾配とPSD surrogate

14.1 合成勾配

\[ G_i = q_{e_i}^\star a_i g_i^{\mathrm{task}} + \lambda_{\mathrm{down}} \frac{\partial\mathcal R_{\mathrm{down}}}{\partial F_i} + \lambda_{\mathrm{factor}} \frac{\partial\mathcal R_{\mathrm{factor}}}{\partial F_i} + \lambda_{\mathrm{turn}} \frac{\partial\mathcal R_{\mathrm{turn}}}{\partial F_i} + \lambda_{\mathrm{var}} \frac{\partial\mathcal R_{\mathrm{var}}}{\partial F_i}. \]

14.2 非対角Hessianの対角majorization

factor penaltyとturnoverはサンプル間結合を持つ。 対称HessianまたはPSD surrogateを \(H\) とし、

\[ D_{ii} = H_{ii} + \sum_{j\neq i}|H_{ij}| + h_{\min}. \]

すると、

\[ D-H\succeq0 \]

である。実際、任意の \(x\) に対して、

\[ x^\top(D-H)x = \frac12 \sum_{i\neq j} |H_{ij}| \left( x_i-\operatorname{sign}(H_{ij})x_j \right)^2 + h_{\min}\|x\|_2^2 \geq0. \]

よって \(D\) は \(H\) の対角上界として利用できる。 ただし非凸な任意項については大域的majorizerとは限らないため、 最終受理にはexact line searchを使用する。

15. Era-Robust Split Criterion 2.0

15.1 ノード統計

\[ G_{A,e} = \sum_{i\in A:e_i=e}G_i, \qquad H_{A,e} = \sum_{i\in A:e_i=e}H_i. \]
\[ G_A=\sum_eG_{A,e}, \qquad H_A=\sum_eH_{A,e}. \]

15.2 coverage-aware leaf prior

era supportはサンプル数ではなく曲率質量で測る。

\[ u_{A,e} = \frac{H_{A,e}} {H_A+\varepsilon}, \qquad N_{\mathrm{eff},A} = \frac{1} {\sum_eu_{A,e}^2+\varepsilon}. \]

親ノードで支持されるera集合を \(\mathcal E_A\) とし、 基準分布をその集合上で再正規化する。

\[ \pi_{A,e} = \frac{ \pi_e\mathbf1[e\in\mathcal E_A] }{ \sum_{k\in\mathcal E_A}\pi_k }. \]

支持集中度を、

\[ \chi_A^2 = \sum_{e\in\mathcal E_A} \frac{ (u_{A,e}-\pi_{A,e})^2 }{ \pi_{A,e}+\varepsilon }. \]

鮮度不足を、

\[ d_A^{\mathrm{stale}} = 1- \sum_e u_{A,e} \exp[-\lambda_d(t_{\max}-t_e)] \]

とし、coverage prior precisionを、

\[ \lambda_{A,\mathrm{cov}} = H_{\mathrm{ref}} \left[ \frac{\kappa_E} {N_{\mathrm{eff},A}+\varepsilon} + \kappa_\chi\chi_A^2 + \kappa_d d_A^{\mathrm{stale}} \right]. \]

総L2 precisionは、

\[ \lambda_{A,\mathrm{total}} = \lambda_2+\lambda_{A,\mathrm{cov}}. \]

15.3 葉値

葉部分問題を、

\[ Q_A(v) = G_Av + \frac12 (H_A+\lambda_{A,\mathrm{total}})v^2 + \lambda_1|v| \]

とすると、最適葉値は、

\[ \boxed{ v_A^\star = - \frac{ \operatorname{SoftThreshold}(G_A,\lambda_1) }{ H_A+\lambda_{A,\mathrm{total}} } } \]
coverage shrinkageは事後的な経験則ではなく、 era coverageに依存するzero-mean Gaussian priorを含む凸葉最適化問題の解になる。

15.4 era別surrogate改善量

親 \(P\) を子 \(L,R\) に分割したとき、

\[ \Delta_e = \left( G_{P,e}v_P+\frac12H_{P,e}v_P^2 \right) - \left( G_{L,e}v_L+\frac12H_{L,e}v_L^2 \right) - \left( G_{R,e}v_R+\frac12H_{R,e}v_R^2 \right). \]

構造正則化とleaf priorの変化は、eraへ恣意的に配分せず、 最後に全体から差し引く。

15.5 Robust lower gain

\[ \mathcal G_{\mathrm{rob}} = \min_{ q\in\mathcal Q_{A,\rho_s,\kappa_s} } \sum_{e\in\mathcal E_P}q_e\Delta_e. \]

capなしの場合の双対は、

\[ \mathcal G_{\mathrm{rob}} = \sup_{\tau_s>0} \left[ -\tau_s\rho_s - \tau_s \log \left( \sum_{e\in\mathcal E_P} \pi_{P,e} \exp(-\Delta_e/\tau_s) \right) \right]. \]

固定temperatureを使う場合は近似soft-minであり、 制約付きDROの厳密値ではない。

15.6 最終split score

\[ \boxed{ \begin{aligned} \operatorname{Score}(P\to L,R) ={}& \mathcal G_{\mathrm{rob}} - \Delta\Omega_{\mathrm{leaf}} - \gamma_{\mathrm{leaf}} \\ &- \kappa_{\mathrm{neg}} \sum_e \pi_{P,e} (-\Delta_e)_+ \\ &- \kappa_{\mathrm{disp}} \operatorname{MAD}_{\pi_P}(\Delta_e). \end{aligned} } \]

\(\operatorname{MAD}_{\pi_P}\) はweighted median absolute deviationであり、 分散よりも極端なera利得へ頑健である。

15.7 採用条件

  • \(\operatorname{Score}>0\)
  • \(N_{\mathrm{eff},L},N_{\mathrm{eff},R}\geq N_{\mathrm{era,min}}\)
  • \(H_L,H_R\geq H_{\min}\)
  • 各子の最近era coverageが下限以上
  • カテゴリ分割がstrict-past統計だけを使用
  • monotonic constraintまたはinteraction constraintに違反しない

15.8 Robust improvement lower bound

各eraで \(\Delta_e\) が真の損失改善量の下界であると仮定する。 これは固定scale Student-t MM surrogateでは構成可能である。 すると、

\[ \mathcal R_{\mathrm{DRO}}(F) - \mathcal R_{\mathrm{DRO}}(F+f) \geq \min_{q\in\mathcal Q} \sum_eq_e\Delta_e. \]

よって \(\mathcal G_{\mathrm{rob}}>0\) は、 separableなDRO task loss部分に対する保守的改善条件になる。 非分離正則化を含む完全objectiveについては、次節のline searchで検証する。

16. Exact Objective Line Search

木 \(f_m\) はPSD surrogateから得た探索方向であり、 そのまま固定learning rateで追加しない。

\[ \alpha_m^\star = \arg\min_{ 0\leq\alpha\leq\alpha_{\max} } \mathcal J \left( F^{(m-1)}+\alpha f_m \right). \]

Armijo条件を、

\[ \mathcal J(F+\alpha f) \leq \mathcal J(F) + c_1\alpha \nabla\mathcal J(F)^\top f, \qquad 0<c_1<1 \]

で確認する。条件を満たさない場合は \(\alpha\) を縮小する。 最小stepでも満たさない場合、その木を棄却する。

この受理機構により、非凸なStudent-t objective、 任意のera variance項、近似対角Hessianを併用しても、 評価した完全objectiveが増加する木は追加されない。

DRO内側問題の \(q^\star\) は各line-search候補で再最適化する。 古い \(q^\star\) の固定評価だけで受理してはならない。

17. Numerai型金融評価との整合

17.1 Era rank correlation

\[ U_{i,e} = \frac{ \operatorname{rank}_e(F_i)-1/2 }{ n_e }, \qquad \widehat F_{i,e} = \Phi^{-1}(U_{i,e}). \]
\[ S_e = \operatorname{Corr}_{W_e} (\widehat F_e,y_e). \]

hard rankは微分不可能であるため、標準構成ではvalidation metricとして使う。 学習objectiveへ入れる場合はsoft-rankを使うが、 計算量と近似誤差を別途管理する。

17.2 Feature-neutral score

\[ F_e^{\mathrm{FN}} = (I-\alpha P_{X,e})C_eF_e, \] \[ S_e^{\mathrm{FN}} = \operatorname{Corr}_{W_e} (F_e^{\mathrm{FN}},y_e). \]

17.3 Meta-model contribution

\[ F_e^{\perp M} = C_eF_e - C_eM_e \frac{ M_e^\top C_e^\top W_eC_eF_e }{ M_e^\top C_e^\top W_eC_eM_e+\varepsilon }. \]
\[ S_e^{\mathrm{MMC}} = \operatorname{Corr}_{W_e} (F_e^{\perp M},y_e). \]

meta-model predictionはout-of-timeで生成されていなければならない。

18. 時系列validation

18.1 Purged walk-forward

\[ \mathcal T_k = \left\{ i: t_i+h_i+\delta_{\mathrm{embargo}}<A_k \right\}, \] \[ \mathcal V_k = \left\{ i: A_k\leq t_i<B_k \right\}. \]

ランダムK-fold、未来eraを含むtarget encoding、 validation eraを用いたscale・factor・category推定は禁止する。

18.2 Validation utility

\[ U_{\mathrm{val}} = \overline S - \kappa_{\mathrm{std}} \operatorname{Std}(S_e) - \kappa_{\mathrm{cvar}} \operatorname{CVaR}_\alpha(-S_e) - \kappa_{\mathrm{mdd}} \operatorname{MDD}(S) \]
\[ \qquad - \kappa_T\operatorname{Turnover} - \kappa_C\operatorname{Cost} - \kappa_F\operatorname{Exposure} + \kappa_{\mathrm{FN}} \overline S^{\mathrm{FN}}. \]

early stoppingは、複数walk-forward foldの中央値または下側信頼限界で判定する。

18.3 不確実性補正

era scoreに自己相関があるため、通常の独立標準誤差を使わない。 block bootstrapまたはHAC分散を用いる。

\[ \operatorname{LCB}_{1-\alpha} = \widehat U - z_{1-\alpha} \widehat{\operatorname{SE}}_{\mathrm{HAC}}(\widehat U). \]

モデル選択では \(\widehat U\) ではなくLCBを最大化する。

19. 完全学習アルゴリズム

  1. target horizon、情報公開時点、embargoを定義し、 各サンプルの利用可能情報集合 \(\mathcal I_i\) を確定する。
  2. purged expanding-window foldを構成する。
  3. strict-pastかつtarget-maturity条件を満たすデータだけで ordered categorical statisticsを生成する。
  4. Huber、Gaussian Student-t、または定数モデルで初期予測 \(F^{(0)}\) を作る。
  5. 学習データだけからfactor operator、portfolio operator、 benchmark loss、scale priorを構築する。
  6. 現在残差から \(\sigma_e,\nu\) を制約付きblock updateする。
  7. era損失 \(L_e\) を計算し、capped KL-DRO内側問題から \(q^\star\) を求める。
  8. task、downside、factor、turnoverの合成勾配を計算する。
  9. Student-t MM曲率、Fisher曲率、PSD penalty Hessianを構成する。
  10. 非対角Hessianを必要に応じて対角majorizeする。
  11. 数値特徴とordered category特徴のヒストグラムを構築する。
  12. 各候補分割についてcoverage-aware leaf priorと葉値を計算する。
  13. era別surrogate改善量 \(\Delta_e\) を計算する。
  14. capped robust lower gain、negative-era penalty、 robust dispersionを含むsplit scoreを評価する。
  15. 制約を満たす最大scoreのsplitをleaf-wiseに採用する。
  16. 木完成後、完全objective上でDROを再最適化しながらline searchする。
  17. objectiveを減少させない木は棄却する。
  18. purged walk-forward validationのLCBが改善しない場合は停止する。

20. 数値安定性の必須条件

  • Student-t CDF、survival probability、Gamma関数はlog-domainで計算する。
  • \(\log\sum\exp\) は最大値を引くlog-sum-expで計算する。
  • \(p\) は \([\varepsilon_p,1-\varepsilon_p]\) に数値上clipする。
  • scaleには \(\sigma_{\min}>0\) を課す。
  • 曲率には \(0<h_{\min}<h_{\max}\) を課す。
  • 因子Gram行列に \(\gamma_ZI\) を加える。
  • category countが少ない場合はglobal priorへ縮小する。
  • DRO計算では \(\pi_e>0\) を保証する。
  • すべてのvalidation変換はtrain fit、validation applyで行う。

24. 2.0最終仕様

\[ \boxed{ \begin{aligned} \mathrm{FORT\mbox{-}GBM\ 2.0} ={}& \mathrm{Student\mbox{-}t\ MM\ Boosting} \\ &+ \mathrm{Target\mbox{-}Maturity\ Ordered\ Learning} \\ &+ \mathrm{Capped\ Era\ KL\mbox{-}DRO} \\ &+ \mathrm{Robust\ Lower\mbox{-}Gain\ Splitting} \\ &+ \mathrm{Coverage\mbox{-}Dependent\ Bayesian\ Leaf\ Prior} \\ &+ \mathrm{PSD\ Factor\ Neutralization} \\ &+ \mathrm{Convex\ Temporal\ Turnover\ Regularization} \\ &+ \mathrm{Exact\ Objective\ Line\ Search}. \end{aligned} } \]

数学的に重要な改訂点は次の通りである。

  1. Student-t曲率を、負Hessianの切り上げではなく、 凹関数の接線上界から導かれるMM majorizerとして再定義した。
  2. 制約付きKL-DROと固定temperature entropic riskを分離した。
  3. era汚染への過集中を防ぐため、敵対重みにcapを加えた。
  4. factor penaltyを固定scaleのPSD二次形式に変更した。
  5. turnoverを固定線形portfolio operator上の凸smooth-L1へ変更した。
  6. 葉縮小を経験的乗算から、coverage-dependent priorを持つ明示的最適化問題へ変更した。
  7. pairwise Student-tを共有Gamma混合変数から厳密に導出した。
  8. ordered statisticsにtarget maturityとembargoを導入した。
  9. 近似木探索の後に完全objectiveを再評価し、 line searchで目的関数増加を拒否する構造を導入した。
  10. 回帰、分類、rankingを共通のera-DRO、ordered learning、 robust split、coverage priorの体系で統一した。
